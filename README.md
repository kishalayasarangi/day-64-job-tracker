# Day 64 — Job Application Tracker

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A full stack job application tracker to manage
internship and job applications with status pipeline.

## How to run
```bash
pip3 install flask
python3 app.py
```
Then open http://localhost:5000

## Features
- Add and track job/internship applications
- Kanban-style status pipeline
- Application stats and insights
- Deadline reminders
- Notes and follow-up tracking
- SQLite database
